alat yang digunakan untuk pengetesan adalah sebuah laptop Lenovo dengan spec Processor AMD Ryzen 3 7320U with Radeon Graphics, 2401 Mhz, 4 Core(s), 8 Logical Processor(s), ram 8 gb, ssd nvme 256gb. 

pengetesan dilakukan dengan alat yang sama, dan sebuah web prototype yang dibuat sendiri dengan teknologi dan framework seperti tailwind, react, node js, juga sudah terconnect dengan GitHub, untuk server adalah server MongoDb dan sudah run sebelum pengetesan dilakukan untuk melihat hasil sebenarnya dalam localhost.
